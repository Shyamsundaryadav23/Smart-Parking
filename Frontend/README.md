# Smart Parking System

## Project Overview

Smart Parking System is a web application that allows users to search, reserve, and manage parking slots in various parking lots. It also includes an admin panel to manage parking lots, slots, and view reservations.

---

## Features

- **User Side:**
  - Register and login
  - View available parking lots and slots
  - Reserve parking slots for a selected time range
  - View personal reservations

- **Admin Side:**
  - Manage parking lots (add, edit, delete)
  - Manage parking slots (mark as available, occupied, reserved)
  - View all reservations

---

## Technologies Used

- **Frontend:**  
  - React  
  - TypeScript  
  - Vite  
  - Tailwind CSS  
  - shadcn-ui (UI components)

- **Backend:**  
  - Node.js  
  - Express.js  
  - dynamoDB

## Getting Started

Follow these steps to run the project locally:

### Prerequisites
- Node.js & npm installed ([Download Node.js](https://nodejs.org/))
- Git installed ([Download Git](https://git-scm.com/))

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/Shyamsundaryadav23/Smart-Parking.git

# 2. Navigate into the project directory
cd smart-parking

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
